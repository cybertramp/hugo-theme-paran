# Hugo theme Paran

Run(on Project root )

```sh
# Build
hugo && cd themes/paran && npm run postbuild && cd ../..

# Run
hugo server --renderStaticToDisk -D
```

# for Debug

```sh
hugo && cd themes/paran && cd ../.. && hugo server --renderStaticToDisk -D
```
